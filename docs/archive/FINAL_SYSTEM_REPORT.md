### 2. **BTC 예측의 본질적 한계**
- 단기(5분, 15분): 예측 가능 ✅
- 중기(1시간): 제한적 예측 가능
- 장기(4시간+): 예측 불가능 (Random Walk)

### 3. **왜 장기 예측이 어려운가?**
