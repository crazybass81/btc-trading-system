## 💡 거래 전략

### 신호 해석
- **신뢰도 >= 70%**: 강한 신호 → 거래 실행
- **신뢰도 65-70%**: 보통 신호 → 소량/주의
- **신뢰도 < 65%**: 약한 신호 → 거래 금지

### 포지션 관리
- **진입**: 신뢰도 70% 이상
- **손절**: -2%
- **익절**: +3%
- **최대 보유**: 4시간

---

## ⚠️ 주의사항

1. **과신 금지**: ML도 100%가 아님
2. **손절 필수**: 항상 -2% 손절선
3. **포지션 제한**: 자본의 5% 이하
4. **시간 제한**: 4시간 내 청산
